How to use that?
----------------

Put `boot.properties` into data/ and `SerializedSystemIni.dat` into data/security/

Cross-domain decryption will not work.


Decryption
----------

Once the two files mentioned above is in place, simply run `make run`:

<pre>
%> make run
username: weblogic
password: z20059864y
</pre>

There you go!
